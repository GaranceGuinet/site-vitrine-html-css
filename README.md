# Site vitrine HTML / CSS

Projet réalisé dans le cadre de mon autoformation en développement web.

## Description
Site vitrine statique composé de plusieurs pages (accueil, à propos, portfolio).
Ce projet m'a permis de pratiquer la structure HTML et la mise en page CSS.

## Pages
- Page d'accueil
- Page À propos
- Page Portfolio

## Technologies utilisées
- HTML5
- CSS3

## Objectifs pédagogiques
Apprendre à structurer un site web et à créer une mise en page responsive en CSS.
Ce projet correspond à mon premier site vitrine réalisé seule dans le cadre de mon autoformation.
